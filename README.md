# UART Transmitter and Receiver in Verilog

Designed and implemented a Universal Asynchronous Receiver Transmitter (UART) communication module in Verilog HDL for serial data transmission and reception. The design includes baud-rate generation, transmitter and receiver modules, start/stop bit handling, and RTL verification through simulation.

## Features

* UART Transmitter (TX)
* UART Receiver (RX)
* Configurable Baud Rate Generator
* Start Bit and Stop Bit Detection
* Error Detection Support
* RTL Simulation and Functional Verification

## Tools Used

* Verilog HDL
* ModelSim Simulator
